## 这是什么

**AI Coding 101** 是面向中国开发者的一站式 AI 辅助编程学习指南。

无论你是刚接触 AI 编程工具的新手，还是想要系统提升效率的老手，这里都有你需要的内容——从工具选型、提示词技巧到团队协作实践，从入门到进阶，一步步带你掌握 AI 编程的核心能力。

## 工具快速对比

| 特性 | Claude Code | Cursor | GitHub Copilot | Codex CLI | Gemini CLI |
|------|-------------|--------|----------------|-----------|------------|
| 类型 | 终端 Agent | IDE（VS Code fork） | IDE 插件 | 终端 Agent | 终端 Agent |
| 模型 | Claude Sonnet/Opus | 多模型可切换 | GPT-4o/Claude | GPT-4o/o3 | Gemini 2.5 Pro |
| 上下文 | 整个项目 | 当前文件 + 引用 | 当前文件 + 引用 | 整个项目 | 整个项目 |
| 最佳场景 | 复杂重构、多文件任务 | 日常编码、快速补全 | 行级补全、轻量辅助 | 自动化脚本、批量任务 | Google 生态集成 |
| 定价模式 | API 按量付费 / Pro $20 / Max $100-200 | 订阅制（$20/40） | 订阅制（$10/19） | API 按量付费 / 随 ChatGPT 订阅 | 免费（有配额） |
| 离线支持 | 否 | 部分 | 部分 | 否 | 否 |
