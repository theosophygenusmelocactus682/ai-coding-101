## What is this

**AI Coding 101** is a one-stop guide for developers learning AI-assisted programming.

Whether you are brand new to AI coding tools or looking to level up your workflow, this guide has you covered — from choosing the right tool and writing effective prompts to team-wide adoption strategies, each chapter takes you one step closer to mastering AI-assisted development.

## Quick Tool Comparison

| Feature | Claude Code | Cursor | GitHub Copilot | Codex CLI | Gemini CLI |
|---------|-------------|--------|----------------|-----------|------------|
| Type | Terminal agent | IDE (VS Code fork) | IDE extension | Terminal agent | Terminal agent |
| Models | Claude Sonnet/Opus | Multi-model | GPT-4o/Claude | GPT-4o/o3 | Gemini 2.5 Pro |
| Context | Entire project | Current file + refs | Current file + refs | Entire project | Entire project |
